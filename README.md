## Running Locally

To view the project locally:

1. Place the project inside your web server directory, such as `htdocs` for XAMPP.
2. Start Apache from XAMPP.
3. Open the project in your browser.

Example:

```text
http://localhost/Iankin%20Portfolio/
```
